1. Project Title
Airbnb Performance Dashboard | Power BI Business Intelligence Dashboard

2. Project Description
The Airbnb Performance Dashboard is an interactive Power BI project developed to analyze Airbnb listings, customer reviews, pricing, ratings, and market performance across multiple cities. The dashboard provides a complete overview of Airbnb's business performance by transforming raw data into meaningful visual insights.

It allows users to compare cities, monitor listing growth, evaluate customer satisfaction, analyze room prices, and identify market trends. The dashboard is designed to help business users make data-driven decisions through easy-to-understand visualizations and interactive reports.

3. Purpose
The main objective of this dashboard is to provide a clear understanding of Airbnb's business performance across different cities. It helps identify the best-performing locations, compare customer ratings, analyze review trends, monitor listing growth, and evaluate pricing for different room types. The dashboard enables stakeholders to make informed business decisions using interactive data visualizations.

5. Data Source
The dashboard is built using an Airbnb dataset containing information about listings, hosts, cities, room types, prices, customer reviews, ratings, and booking-related details. The dataset was cleaned, transformed, and modeled in Power BI before creating the visualizations.

7. Dashboard Insights
Paris, New York, and Sydney contribute a significant share of total Airbnb listings and customer reviews.
Hotel rooms have the highest average price, while private rooms are the most affordable accommodation option.
Customer ratings remain consistently high across all cities, indicating a positive guest experience.
Airbnb listings experienced rapid growth before slowing down due to regulatory changes and the COVID-19 pandemic.
European cities such as Paris and Rome receive higher review activity during the summer months, while New York experiences increased reviews during the holiday season.
